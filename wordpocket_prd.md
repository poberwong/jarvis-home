# WordPocket 产品需求文档（PRD）v1.0

**版本**：1.0  
**日期**：2026-02-18  
**状态**：Draft

---

## 1. 产品概述
**愿景**：让高频阅读外文资料的人，以极低成本持续积累领域生词并体系化复习。  
**价值主张**：**“阅读即收集，通勤即复习”**。  
**解决的问题**：翻译工具只能临时理解，无法形成可复用的个人词汇体系。

## 2. 用户研究
**目标用户**：希望在技术/金融/科研等垂直领域快速提升英语的人群。  
**痛点**：阅读时约 25% 领域词影响理解；翻译解决当前，学习无沉淀。  
**需求优先级（MVP）**：
- 划词一键入库 ✅
- 生词自动场景分类 ✅
- 生词卡片（上下文/音标/翻译/读音）✅
- 端侧复习（通勤/碎片）✅

## 3. 核心功能
### 3.1 输入端：浏览器插件
- 划词一键入库
- 自动记录 source URL
- 采集上下文段落

### 3.2 AI 自动处理
- 语境分类（科技/人文/医学/生活等）
- 生成翻译与解释
- 抽取上下文为 Context

### 3.3 输出端：手机端
- 按场景切换生词列表
- 生词卡片：上下文/音标/翻译/读音
- 数据清洗与偏好调整
- 版本切换（不同主题词库）

## 4. 技术架构（建议）
- 插件采集 → API → AI 处理 → 词库 DB → 移动端展示
- 技术栈建议：
  - 插件：Chrome Extension
  - 后端：FastAPI / Node
  - DB：PostgreSQL + Redis
  - AI：LLM + 语音 TTS

## 5. UI/UX 设计
- 1 步入库、3 步内复习
- 卡片化展示 + 标签切换
- 场景优先、记忆负担最低

## 6. 开发路线图
**MVP（8周）**：插件采集 + AI 分类 + 生词卡片 + 场景切换  
**v1.1**：错词清洗/用户偏好  
**v1.2**：复习提醒/艾宾浩斯  
**v2.0**：多语言支持 + 社区分享

## 7. 商业模式
- 免费版：限量入库 + 基础复习
- Pro：无限入库 + AI 优化 + 高级复习
- 企业版：团队词库 + 培训/行业场景包

## 8. 成功指标
- 北极星：每周有效复习次数
- 转化率 / 留存 / 复习完成率

## 9. 风险与挑战
- AI 误分类 → 允许用户纠错
- 采集合规 → 只保存必要文本 + 提示用户来源
- 冷启动词库不足 → 预置通用词库

## 10. 附录
- 竞品：传统生词本、翻译软件、AI 阅读工具
